# Self-Hosted

This document provides a guide for deploying PenX with Docker.

The PenX service is packaged using Docker and published to Docker Hub at [hub.docker.com/r/penxio/penx](https://hub.docker.com/r/penxio/penx).

## Docker Run

To start PenX service with docker run, execute the following one command:

```bash
docker run -d \
  --name penx-self-hosted \
  -e DATABASE_URL={{postgresql_database_url}} \
  -p 3000:3000 \
  penxio/penx-postgresql:latest
```

A real world example:

```bash
docker run -d \
  --name penx-self-hosted \
  -e DATABASE_URL=postgresql://postgres:a73F4EXMWYpFxpxH@db.xwjwmhepndygekdrxasp.supabase.co:5432/postgres \
  -p 3000:3000 \
  penxio/penx-postgresql:latest
```

If you use MySQL, use image: `penxio/penx:mysql-latest`.

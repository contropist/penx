# Release an Extension

Releasing an extension is straightforward with PenX.

- Quickly, One command to release by `penx release`.
- Freedom, No limit, No review, similar to publishing an npm package.

## How to release?

Suppose you are developing an extension named extension-template-minimal. To release it, simply execute the following command:

```bash
cd extension-template-minimal
npm run release
```

Once released, your extension will be available for search and installation in the PenX marketplace.

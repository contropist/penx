# Custom UI React

This guide walks you through creating a custom UI Command App using [React](https://react.dev/).

## Start with React template

Clone the sample plugin using Git.

```bash
git clone git@github.com:penxio/extension-template-react.git
```

## Develop extension

After cloning the repository, navigate to the project directory and install the dependencies:

```bash
cd penx-extension-react
npm install # install the dependencies
npm run dev # compile the source code
```

Running npm run dev will compile the source code and automatically load the sample extension in PenX.

Open the PenX Command palette, and you will see two commands:

![commands](/images/commands-react.png)

## Important files

The project consists of the following key files:

- `manifest.json`
- `hello.command.tsx`
- `count.command.tsx`

### `manifest.json`

The manifest.json file contains metadata about the project.

```json
{
  "name": "extension-template-react",
  "title": "Extension Template React",
  "description": "This is a custom UI demo.",
  "icon": {
    "name": "mdi--react",
    "className": "bg-blue-500"
  },
  "commands": [
    {
      "name": "hello",
      "title": "Hello React",
      "icon": {
        "name": "mdi--react",
        "className": "bg-blue-500"
      },
      "mode": "custom-ui",
      "framework": "react"
    },
    {
      "name": "count",
      "icon": {
        "name": "lucide--clock",
        "className": "bg-gradient-to-r from-teal-500 to-cyan-500"
      },
      "title": "React Count",
      "mode": "custom-ui",
      "framework": "react"
    }
  ]
}
```

The **command name** is associated with the `[commandName].command.tsx` file, which is a very important convention.

- hello -> `src/hello.command.tsx`
- count -> `src/count.command.tsx`

### `hello.command.tsx`

The `hello.command.tsx` file will render the "Hello World" string:

```tsx
export function Main() {
  return (
    <div className="flex justify-center items-center h-full">
      <h1 className="text-4xl">Hello World!</h1>
    </div>
  )
}
```

![hello](/images/custom-ui-hello.png)

### `count.command.tsx`

The `count.command.tsx` file shows a basic counter demo:

```tsx
import { useState } from 'react'

export function Main() {
  const [count, setCount] = useState(0)

  return (
    <div className="flex justify-center items-center h-full bg-gradient-to-r from-cyan-500 to-blue-500">
      <button
        className="outline-none p-3 text-lg rounded-3xl bg-blue-300 hover:opacity-70 "
        onClick={() => {
          setCount(count + 1)
        }}
      >
        Count is {count}
      </button>
    </div>
  )
}
```

## Style

[Tailwind CSS](https://tailwindcss.com) is supported out of the box with no configuration needed.

Of course, you can choose to use plain CSS or CSS-in-JS, but we recommend using Tailwind CSS for its simplicity and efficiency.

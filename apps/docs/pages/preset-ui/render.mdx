# Render

Render some component to Command App.

## Exapmple

### Render Markdown

Render a markdonwn string.

```ts
import { MarkdownBuilder, render } from '@penxio/api'

export async function main() {
  render(new MarkdownBuilder('Hello **world**'))
}
```

### Render List

Render a basic list with title.

```ts
import { ListBuilder, render } from '@penxio/api'

export async function main() {
  const list = new ListBuilder([
    { title: 'First Item' },
    { title: 'Second Item' },
    { title: 'Third Item' },
  ])
  render(list)
}
```
